# Calculadora Web

Este é um projeto simples de uma calculadora desenvolvida utilizando **HTML, CSS e JavaScript**.

## Objetivo

O objetivo do projeto foi praticar conceitos fundamentais do desenvolvimento web, como:

- Estruturação de páginas com HTML
- Estilização com CSS
- Manipulação do DOM com JavaScript
- Lógica de programação para operações matemáticas

## Funcionalidades

- Operações básicas (+, -, *, /)
- Interface interativa com botões
- Suporte ao teclado
- Limpar e apagar números

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
